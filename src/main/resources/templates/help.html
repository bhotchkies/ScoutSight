How To
Working Directory

Place your input CSV files in a directory and run ScoutSight from that location. Output HTML reports will be written to an output/ subfolder in the same directory.

Advancement CSV
1
Go to advancements.scouting.org
2
Click Reports in the left navigation
3
Scroll down to Quick Exports
4
Find the Advancements row and click Export
5
Save the downloaded .csv file and pass it as the first argument to ScoutSight
Scouts CSV — Optional

Adds patrol and grade information from Scoutbook to the reports.

1
Go to advancements.scouting.org
2
Click Reports in the left navigation
3
Scroll down to Quick Exports
4
Find the Scout/Members row and click Export
5
Save the downloaded .csv file and pass it as the second argument to ScoutSight
